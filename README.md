# AI Academic Workflows Toolkit

Open-source toolkit for AI-assisted academic research, teaching workflows, and reproducible economic analysis.

## Scope
- AI-assisted literature review and text analysis
- Prompt-based research workflows
- Paper-Canvas templates
- Reproducible research pipelines (data processing, estimation)
- Teaching integration (university-level courses)

## Use Cases
- Academic writing with AI support
- Economic policy analysis
- Reproducible empirical research
- AI integration in higher education

## Context
Developed and used in courses at the University of Hamburg (IGK1, IGK2).
Supporting materials: ~2000+ downloads via Zenodo (OER).

## Roadmap
- Automated PR review workflows
- Data pipeline validation
- Integration with external APIs (LLMs, research tools)
